# platform.kidjig

KidJig platform — product frontend/backend for the KidJig project.

## Stack

Next.js, TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/platform.kidjig.git
cd platform.kidjig
npm install
npm run dev   # or: npm start
```
